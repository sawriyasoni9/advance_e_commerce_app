# Splash Screen Assets

## Character Image

Place your splash screen character illustration here:

**File name:** `splash_character.png`

**Recommended dimensions:** 
- Width: 500-750px (will scale to fit)
- Height: 700-1050px (will scale to fit)
- Format: PNG with transparency

The image should be the 3D character illustration of the woman with shopping bags as shown in your design.

If the image is not found, the splash screen will show a placeholder with instructions.
